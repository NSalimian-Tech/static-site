Let's decode it line by line: | Line | What it means | |---|---| | `` | "Hey browser, this is a modern HTML page" | | `` | The root of everything. `lang="en"` = English | | `` | Invisible settings and links — not shown on page | | `` | Supports all characters (including emoji 🎉) | | `` | Makes page look right on phones | | `` | The text shown in the browser tab | | `<link rel="stylesheet"...>` | Connects your CSS file | | `<script src="..." defer>` | Connects your JS file (after page loads) | | `<body>` | **Everything visible on the page lives here** | --- ## 🧱 Step 2 — Add Your Profile Content Now let's fill the `<body>` with content. Replace the comment with this: <div class="file-label">📄 index.html — inside <body></div> ```html <body> <div id="profile-card"> <h1 id="explorer-name">Commander Troy</h1> <p class="explorer-title">Astronaut in Training 🚀</p> <ul class="explorer-stats"> <li>🪐 Favourite planet: <strong>Saturn</strong></li> <li>🌙 Moon landings: <strong>0 (so far!)</strong></li> <li>⭐ Skills: <strong>HTML · CSS · JavaScript</strong></li> </ul> <button id="launch-btn">✨ Launch Mission</button> <p id="mission-status"></p> </div> </body> ``` **Save and look at your browser** — raw and unstyled, but it's *your* content on a real webpage! --- ## 🧱 Step 2 — Understanding the Tags We Used | Tag | What it is | |---|---| | `<div id="profile-card">` | A container box for our whole card | | `<h1>` | The biggest heading — your name | | `<p>` | A paragraph of text | | `<ul>` | An **unordered list** (bullet points) | | `<li>` | A **list item** inside the `<ul>` | | `<strong>` | Bold text — for emphasis | | `<button>` | A clickable button | | `<p id="mission-status">` | An empty paragraph — JS will fill it later! | > 💡 Notice how we used both `id` and `class`: > - `id="profile-card"` → **unique**, only one card > - `class="explorer-stats"` → could be reused on multiple lists --- ## 🎨 Step 3 — Add the Cosmic Styling Time to make it beautiful! Open **`style.css`** and add this: <div class="file-label">📄 style.css</div> ```css /* Import a cool font from Google */ @import url('https://fonts.googleapis.com/css2?family=Orbitron:wght@700&family=Exo+2&display=swap'); Let's decode the new CSS concepts: | CSS | What it does | |---|---| | `@import url(...)` | Loads a font from Google Fonts | | `* { box-sizing: border-box }` | Makes sizing predictable (always include this!) | | `min-height: 100vh` | Page is at least as tall as the screen (`vh` = viewport height) | | `display: flex; justify-content: center` | Centers the card on the page | | `rgba(255,255,255, 0.05)` | White with only 5% opacity — semi-transparent | | `border-radius: 20px` | Rounded corners | | `box-shadow: 0 0 60px ...` | Glowing shadow | | `transition: transform 0.2s` | Smooth animation when something changes | | `transform: scale(1.05)` | Makes element 5% bigger on hover | > **Save and check your browser** — you should have a glowing space card! ✨ --- ## ⚡ Step 4 — Add JavaScript Interactivity The button exists but does nothing. Let's fix that! Open **`script.js`**: <div class="file-label">📄 script.js</div> ```javascript // Step 1: Find the elements we want to work with const launchBtn = document.getElementById("launch-btn"); const missionStatus = document.getElementById("mission-status"); const explorerName = document.getElementById("explorer-name"); // Step 2: Listen for a click on the button launchBtn.addEventListener("click", function() { launchMission(); }); // Step 3: Define what happens when the button is clicked function launchMission() { // Change the button text launchBtn.textContent = "🛸 Mission Active!"; // Add text to the empty paragraph missionStatus.textContent = "📡 Signal received. Destination: Saturn orbit."; // Change the style of that paragraph missionStatus.style.color = "#7effd4"; missionStatus.style.marginTop = "20px"; missionStatus.style.fontSize = "15px"; } ``` **Save it and click the button in your browser!** 🎉 --- ## ⚡ Step 4 — JavaScript, Line by Line ```javascript const launchBtn = document.getElementById("launch-btn"); ``` → **"Find the element with `id="launch-btn"` and store it in a variable called `launchBtn`"** `const` = a variable that won't be reassigned. Think of it as a labelled box. --- ```javascript launchBtn.addEventListener("click", function() { launchMission(); }); ``` → **"When `launchBtn` is clicked, run the `launchMission` function"** `addEventListener` = attach an ear to listen for a specific event. --- ```javascript launchBtn.textContent = "🛸 Mission Active!"; ``` → **"Change the text inside the button"** ```javascript missionStatus.style.color = "#7effd4"; ``` → **"Apply a CSS style directly from JavaScript"** --- ## 🎨 Step 5 — Style the Mission Status Add this to your **`style.css`** to polish the status message: <div class="file-label">📄 style.css (add at the bottom)</div> ```css /* The mission status message (starts empty, JS fills it) */ #mission-status { font-style: italic; min-height: 24px; /* reserves space even when empty */ transition: all 0.4s; /* smooth fade when JS changes it */ } ``` --- And add one more touch to **`script.js`** — add a CSS *class* instead of inline styles: <div class="file-label">📄 script.js — update launchMission()</div> ```javascript function launchMission() { launchBtn.textContent = "🛸 Mission Active!"; launchBtn.disabled = true; // prevent clicking twice! missionStatus.textContent = "📡 Signal received. Destination: Saturn orbit."; // Add a class (cleaner than inline styles!) missionStatus.classList.add("status-active"); } ``` ```css /* style.css — add this too */ .status-active { color: #7effd4; margin-top: 20px; font-size: 15px; animation: fadeIn 0.5s ease; } @keyframes fadeIn { from { opacity: 0; transform: translateY(6px); } to { opacity: 1; transform: translateY(0); } } ``` --- ## 🧩 Bonus — Make the Name Editable! Want to personalise it? Let's make the name **clickable to edit**! <div class="file-label">📄 index.html — update the h1</div> ```html <h1 id="explorer-name" title="Click to change your name!"> Commander Troy </h1> ``` <div class="file-label">📄 script.js — add this below your existing code</div> ```javascript // Make the name editable on double-click explorerName.addEventListener("dblclick", function() { const newName = prompt("Enter your explorer name:"); if (newName) { // only update if they typed something explorerName.textContent = newName; } }); ``` <div class="file-label">📄 style.css — add this</div> ```css #explorer-name { cursor: pointer; } #explorer-name:hover { opacity: 0.85; } ``` > **Double-click your name in the browser to rename yourself!** 🧑‍🚀 --- ## 🗂️ The Complete File Overview Here's what your final files look like together: ``` space-explorer/ │ ├── index.html ← Structure: div, h1, p, ul, li, button │ Attributes: id, class, title │ Head: meta charset, viewport, link, script │ ├── style.css ← Selectors: #id, .class, element, * │ Properties: color, background, padding, border-radius │ Layout: flexbox (display, justify-content, align-items) │ Effects: box-shadow, transition, transform, @keyframes │ Font: @import from Google Fonts │ └── script.js ← Variables: const DOM: getElementById, textContent, style, classList Events: addEventListener("click"), addEventListener("dblclick") Logic: if statement Functions: function declaration and call UI: prompt(), disabled property ``` --- ## 🔍 Debugging Tips — When Things Go Wrong Don't panic! Every developer debugs daily. Here's your toolkit: **1. Nothing showing up?** → Check your file is saved (`Ctrl+S`) → Make sure Live Server is running (click **Go Live**) **2. Styles not applying?** → Open browser DevTools: press **`F12`** → click the **Elements** tab → Click any element to see what CSS is applied (and what's being overridden) **3. JavaScript not working?** → F12 → **Console** tab — red text = an error with a line number → Add `console.log("am I running?")` anywhere in your JS to test **4. Typo hunt:** → Missing `"` or `>` or `}` breaks everything → VS Code highlights errors — look for red underlines! > 💜 **Every error message is a clue, not a failure.** Read it, Google it, fix it. --- ## 🚀 What You Just Learned You built a real webpage from scratch and used every concept from the previous guide! | Concept | Where you used it | |---|---| | HTML skeleton | `<!DOCTYPE>`, `<head>`, `<body>` | | Tags | `div`, `h1`, `p`, `ul`, `li`, `button` | | Attributes | `id`, `class`, `title`, `defer` | | `<head>` links | `<link>` for CSS, `<script>` for JS | | Meta tags | `charset`, `viewport` | | CSS selectors | `#id`, `.class`, element, `*` | | Box model | `padding`, `margin`, `border`, `width` | | CSS transitions | `transition`, `transform`, `@keyframes` | | JS DOM | `getElementById`, `textContent`, `style` | | JS events | `addEventListener("click")` | | JS functions | `function launchMission() { }` | --- <!-- _class: center-slide --> # 🌌 Mission Complete! ### You built a webpage.
